# Lido DAO Outreach Research

**Prospect Value:** $25-40K
**Confidence:** 7/10
**Research Date:** 2026-02-04

## Lido DAO Overview
- **Protocol:** Liquid staking protocol (ETH, stETH, ~$30B+ TVL)
- **Governance:** Aragon-based DAO, LDO token holders
- **Scale:** Largest liquid staking protocol, multi-chain

## Pain Points Identified

1. **Multi-chain Governance Complexity**
   - Lido operates on Ethereum, but governance discussions span multiple chains
   - Cross-chain proposals require coordination across ecosystems
   - Monitoring governance sentiment across chains = manual effort

2. **Node Operator Tracking**
   - 30+ node operators across different chains
   - Performance monitoring, key rotation, compliance tracking
   - Current process: Spreadsheets + manual reporting

3. **Community Sentiment Analysis**
   - Large Discord community (50K+ members)
   - Twitter sentiment moves LDO price
   - Proposal discussions span multiple forums (Snapshot, Aragon, Discord)

4. **Delegation & Voting Power**
   - LDO holders delegate to active delegates
   - Tracking delegate voting patterns = manual
   - Identifying aligned delegates requires research

## Outreach Strategy

- Follow key delegates on Twitter (@ligi, @Island_ADC, etc.)

### Phase 2: Value-First DM (Day 4)
- **Target:** Governance moderators or active delegates

### Phase 3: Pilot (Days 5-11)
- Build lightweight tracker:
  - Snapshot proposals feed
  - Aragon on-chain voting monitor
  - Key delegate voting pattern alerts
- Daily brief: "What governance happened today"
