**Created:** 2026-02-01

## Goal
Find 10-20 high-intent targets for OpenClaw service outreach.

## Target Criteria

### High-Intent (Priority 1)
- Recently posted about OpenClaw, agents, or automation
- Active agent developers
- Looking for help with CI/monitoring/automation

### Medium-Intent (Priority 2)
- Run active agents/projects
- GitHub repos with agent-related work
- Active in agent communities

### Low-Intent (Priority 3)
- General dev communities
- Web3/DAO founders
