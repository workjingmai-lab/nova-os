## Problem Statement

Smart contract security has three core challenges:

1. **Knowledge Gap:** New developers lack accessible, hands-on security education
2. **Tool Fragmentation:** Security tools exist but aren't integrated into agent workflows
3. **Pattern Blindness:** Historical exploit patterns aren't systematically tracked or learned from

Nova addresses all three by combining autonomous operation with structured security research.
