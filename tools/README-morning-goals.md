# morning-goals.py

**Auto-generate daily goals based on context and momentum.**

## What It Does

Generates 3-5 focused goals for the day by analyzing:
- Active long-term goals (from goals/active.md)
- Recent diary activity (last 24h achievements)
- Yesterday's incomplete items
- Default fillers if needed

Creates/overwrites `today.md` with formatted goals.

## Usage

```bash
# Generate today.md (backs up existing)
python3 tools/morning-goals.py

# Preview without writing
python3 tools/morning-goals.py preview
```

## Dependencies

- Python 3.7+
- Standard library only

## File Dependencies

**Reads from:**
- `diary.md` â€” Recent activity, achievements
- `goals/active.md` â€” Long-term incomplete goals
- `.agent_state.json` â€” Heartbeats, goals completed

**Writes to:**
- `today.md` â€” Daily goals (overwrites, backs up existing)

## Goal Selection Logic

1. **1-2 active long-term goals** â€” Random sample from active.md
2. **1 momentum continuation** â€” Based on recent achievement
3. **1 incomplete item** â€” Yesterday's pending tasks
4. **Fill to 4-5 goals** â€” Random defaults (learn, document, engage, improve, build)

## Use Cases

- **Morning planning** â€” Start day with focus
- **Decision fatigue elimination** â€” No "what should I do today?"
- **Context-aware** â€” Builds on recent work
- **Automatic backups** â€” Preserves yesterday's today.md

## Integration

Pairs well with:
- `evening-reflection.py` â€” End day with review
- `session-starter.py` â€” New session context
- `today.md` â€” Source for other tools

## Customization

Edit defaults in `generate()`:
```python
defaults = [
    "Learn one new thing today",
    "Document something you built",
    "Engage with another agent",
    "Review and improve one existing tool",
    "Build something small but complete"
]
```

Adjust goal count:
```python
goals = self.generate(count=5)  # Instead of default 4
```
