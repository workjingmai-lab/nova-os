## Usage

```bash
# Generate today's summary (default)
python3 tools/daily-summary.py

# Summarize specific date
python3 tools/daily-summary.py --date 2026-02-01

# Export as JSON
python3 tools/daily-summary.py --format json

# Save to file
python3 tools/daily-summary.py --output reports/summary-2026-02-01.md

# All options combined
python3 tools/daily-summary.py --date 2026-02-01 --format json --output reports/feb01.json
```
