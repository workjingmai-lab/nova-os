# agent-logger.py

**Purpose:** Lightweight work-tracking system for AI agents ‚Äî log tasks, track time, generate daily reports.

## What It Does

- **Track active tasks** ‚Äî Start/stop task tracking with automatic time logging
- **Measure duration** ‚Äî Calculates how long each task takes
- **Daily reports** ‚Äî Generate formatted summaries of your work
- **Minimal dependencies** ‚Äî Stdlib only, no external packages required

## When to Use It

**For agents who want to:**
- Track how long tasks actually take (vs. estimated)
- Generate daily work reports automatically
- Maintain a lightweight work log without complex infrastructure
- Measure productivity across different task types

## Usage

```bash
# Start a task
python3 tools/agent-logger.py --start "Documentation sprint"

# Complete the current task
python3 tools/agent-logger.py --done "Created 5 READMEs"

# Show current status
python3 tools/agent-logger.py --status

# Generate daily report
python3 tools/agent-logger.py --report
```

## Data Storage

- **State file:** `~/.agent-logs/state.json` ‚Äî Current task + today's completed tasks
- **Daily logs:** `~/.agent-logs/work-YYYY-MM-DD.md` ‚Äî Markdown log entries
- **Automatic cleanup:** Old daily logs persist for archival

## State Format

```json
{
  "current_task": {
    "name": "Documentation sprint",
    "started": "2026-02-02T13:20:00Z"
  },
  "tasks_today": [
    {
      "name": "blocker-tracker documentation",

}
```

## Why It Matters

**Measure to improve.** This logger helps you:
- **Track actual vs. estimated time** ‚Äî Are your time estimates accurate?
- **Identify slow tasks** ‚Äî Which tasks take longer than expected?
- **Generate reports** ‚Äî Show stakeholders what you accomplished
- **Optimize workflow** ‚Äî Data-driven decisions about task allocation

**For autonomous agents:** A simple way to prove productivity with time-stamped, auditable logs.

## Integration

- **Daily review:** Run `--report` every evening for work summary
- **Task estimation:** Compare estimated vs. actual time to improve planning
- **Productivity tracking:** Use with `agent-productivity-score.py` for comprehensive metrics
- **Stakeholder reporting:** Share daily reports for transparency

## Example Workflow

```bash

python3 tools/agent-logger.py --start "Review grant submissions"

# Complete task when done
python3 tools/agent-logger.py --done "5 grants ready, needs GitHub push"

# Start next task
python3 tools/agent-logger.py --start "Moltbook engagement"
```

## Configuration

Edit these variables at the top of the script:
```python
LOG_DIR = Path.home() / ".agent-logs"  # Where to store logs
STATE_FILE = LOG_DIR / "state.json"     # Current task state
```
