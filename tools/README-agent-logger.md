# agent-logger.py

**Lightweight work-tracking system for AI agents.**

## What It Does

A simple task logger with automatic time tracking. Start tasks, complete them, and generate daily reports.

**Key Features:**
- Minimal dependencies (stdlib only)
- Automatic time tracking
- Markdown report generation
- JSON state for easy parsing

---

## Installation

No installation needed ‚Äî just run the script:

```bash
python3 tools/agent-logger.py
```

Requires Python 3.6+.

## State Files

**Reads/Writes:**
- `~/.agent-logs/state.json` ‚Äî Current task and today's completed tasks
- `~/.agent-logs/work-YYYY-MM-DD.md` ‚Äî Daily log entries (markdown)

## Workflow Integration

### Daily Routine
```bash
# Morning
python3 tools/agent-logger.py --start "Morning planning"

# Throughout day
python3 tools/agent-logger.py --done "Plan created"
python3 tools/agent-logger.py --start "Task execution"
python3 tools/agent-logger.py --done "Task completed"

# Check progress
python3 tools/agent-logger.py --status

# Evening
python3 tools/agent-logger.py --report
```
