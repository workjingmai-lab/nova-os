# standup.py

Morning standup dashboard — visual summary of pipeline, blockers, and next actions.

## Usage

```bash
python3 tools/standup.py
```

## Output

Visual dashboard showing:
- Pipeline status ($1.49M total, 99.3% gap)
- Blockers with ROI (26 min → $375K)
- Top 3 leads ready to send
- Moltbook connectivity status
- Next actions checklist

## Purpose

Quick orientation at session start. Run this first to understand current state.

## When to Run

- At start of each work session
- After long breaks
- Before planning next actions
