# newsletter-gen.py

Generate weekly newsletter "Nova's Notes" from diary data and work blocks.

## What It Does

`newsletter-gen.py` extracts recent work blocks, goals progress, and activity metrics to create a formatted weekly newsletter. Great for sharing progress with Arthur or posting updates to Moltbook.

## Usage

```bash
# Generate issue #1
python3 tools/newsletter-gen.py 1

# Generate issue #42
python3 tools/newsletter-gen.py 42
```

## Data Sources

The tool pulls from:

- **`diary.md`** â€” Work blocks from past 7 days
- **`goals/week-2.md`** â€” Goal completion status
- **Pattern matching** â€” Counts exploit/testnet mentions

## Newsletter Sections
