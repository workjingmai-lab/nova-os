# README — grant-submit.py

## What It Does
Automates grant application submissions for 5 major Web3 grant platforms.

**Generates ready-to-submit content for:**
- Gitcoin ($10K-$150K)
- Optimism RPGF ($10K-$100K)
- Octant ($10K-$100K)
- Olas ($10K-$50K)
- Moloch DAO ($10K-$50K)

**Total potential: $130K+**

## When to Use
**When your grant pipeline is ready.** Run this after:
- GitHub repo is public
- README.md exists
- Tools are documented
- Metrics are tracked

## Usage

### Check Prerequisites First
```bash
python3 tools/grant-submit.py --check
```
Checks:
- ✅ GitHub repo configured
- ✅ GitHub auth (CLI or SSH)
- ✅ README.md exists

### Submit All Grants
```bash
python3 tools/grant-submit.py --all
```
Generates submission files for all 5 platforms in `tmp/grant-submissions/`

### Submit Specific Grant
```bash
python3 tools/grant-submit.py gitcoin
python3 tools/grant-submit.py optimism
python3 tools/grant-submit.py octant
python3 tools/grant-submit.py olas
python3 tools/grant-submit.py moloch
```

### Preview Without Submitting
```bash
python3 tools/grant-submit.py --dry-run
```
Shows what will be generated without writing files.

## How It Works

1. **Validates prerequisites** — GitHub, auth, README
2. **Loads pipeline data** — Current metrics from revenue-pipeline.json
3. **Generates submission** — Platform-specific content with impact metrics
4. **Saves to file** — JSON in `tmp/grant-submissions/`
5. **Provides instructions** — URL + copy/paste content

## Troubleshooting

**❌ "Unknown grant"**
- Check grant name matches: gitcoin, optimism, octant, olas, moloch

**❌ "Prerequisites not met"**
- Run `--check` to see specific failures
- Fix GitHub auth: `gh auth login` OR set up SSH keys
- Create README.md in workspace root

**❌ Wrong repository URL**
- Check `git remote -v` shows correct GitHub URL
- Tool auto-converts SSH → HTTPS format
