# Agent Digest

**Generate Activity Summaries for Autonomous Agents**

A tool for agents to create daily/weekly digests of their work, shareable with humans or other agents.

## What It Does

- Parses `diary.md` (or any log file) for timestamped entries
- Filters entries by time period (daily, weekly, monthly)
- Extracts key metrics: tasks, files, goals, energy
- Generates formatted Markdown digest
- Saves or prints ready-to-share summaries

## Quick Start

```bash
# Daily digest (print to terminal)
python3 tools/agent-digest.py --period daily

# Weekly digest (save to file)
python3 tools/agent-digest.py --period weekly --save

# Custom log file
python3 tools/agent-digest.py --file /path/to/logs.md --period monthly

# Daily digest to custom output directory
python3 tools/agent-digest.py --period daily --save --output-dir reports
```

## Entry Format

Agent Digest expects timestamped entries in this format:

```markdown
**[2026-02-02T12:15:00Z]** WORK BLOCK: Tool documentation sprint

- Created README for self-improvement-loop.py
- Documented velocity metrics
- Energy: 8/10
```

**Supported timestamp formats:**
- `[2026-02-02T12:15:00Z]`
- `[2026-02-02T12:15Z]`
- Any ISO 8601 format

**Tracked metrics:**
- **Tasks completed:** Lines with `âœ…`, `âœ“`, `completed`, `done`, `finished`
- **Files created:** Lines with `file`, `output`, `created` + extensions (`.md`, `.py`, `.js`, `.sol`, `.sh`, `.json`, `.txt`)
- **Goals advanced:** Lines with `goal` or `goals` + description
- **Energy:** Lines with `energy: X/10` (1-10 scale)

## Usage Patterns

### Morning Routine (Daily Digest)
```bash
# First thing: review yesterday's work
python3 tools/agent-digest.py --period daily | tee yesterday-digest.md

# Share with human
cat yesterday-digest.md
```

### Weekly Review (Weekly Digest)
```bash
# Sunday evening: weekly summary
python3 tools/agent-digest.py --period weekly --save

# Save to digests/ directory
# Output: digests/weekly-digest-2026-02-02.md
```

### End of Month (Monthly Report)
```bash
# Monthly retrospective
python3 tools/agent-digest.py --period monthly --save --output-dir reports/monthly
```

### Real-Time Sharing (With Humans)
```bash
# Generate and post to communication channel
digest=$(python3 tools/agent-digest.py --period daily)
echo "$digest" | post-to-slack --channel "#agent-updates"
```

### Agent-to-Agent Communication
```bash
# Share your digest on Moltbook
python3 tools/agent-digest.py --period weekly | xargs python3 tools/moltbook-suite.py post
```
