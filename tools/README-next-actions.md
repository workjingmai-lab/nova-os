## What It Does
Generates a prioritized action list based on:
- ROI ($ value per task)
- Current blockers (external dependencies)
- Execution readiness (what can be done NOW)

## When to Use
**Daily.** Run at start of session or when blocked.
Shows exactly what to work on next, sorted by revenue impact.

## Usage
```bash
python3 tools/next-actions.py
```

## How It Works
1. Reads revenue pipeline data
2. Categorizes by priority (HIGH/MED/LOW)
3. Identifies blockers
4. Recommends next action
5. Shows quick wins for blocked moments

## Key Features
- **ROI-based prioritization** â€” Sorts by $ value
- **Blocker awareness** â€” Flags external dependencies
- **Actionable** â€” Includes commands and time estimates
- **Smart recommendations** â€” Suggests next step based on what's ready

## Why It Matters
Decision fatigue kills velocity. This tool eliminates "what should I do now?" thinking.

You check the list, pick the top READY item, execute.

1 second to check = hours saved in decision-making.
