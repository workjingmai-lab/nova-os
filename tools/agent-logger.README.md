# agent-logger.py

**Lightweight work-tracking system for AI agents.**

## What It Does

Tracks agent work with automatic time tracking:
- **Start/complete tasks** ‚Äî Log work with timestamps
- **Duration tracking** ‚Äî Auto-calculates task duration
- **Daily reports** ‚Äî Generate formatted markdown reports
- **JSON state** ‚Äî Easy parsing for analytics

**Value:** Creates structured work log for self-monitoring and performance analysis.

## Usage

### Start a Task
```bash
python3 tools/agent-logger.py --start "Write README for tool-x"
```
Logs task start time, marks as active.

### Complete a Task
```bash
python3 tools/agent-logger.py --done "Created 3,500-byte README with examples"
```
Calculates duration, saves to daily log.

### Show Status
```bash
python3 tools/agent-logger.py --status
```
Displays current task and today's progress.

### Generate Daily Report
```bash
python3 tools/agent-logger.py --report
```
Prints formatted daily summary.

    "name": "Write README for tool-x",
    "started": "2026-02-03T14:10:00Z"

  "tasks_today": [

## Configuration

Modify these at the top of the file:
```python
LOG_DIR = Path.home() / ".agent-logs"
STATE_FILE = LOG_DIR / "state.json"
DAILY_LOG = LOG_DIR / f"work-{datetime.now().strftime('%Y-%m-%d')}.md"
```

## API Reference

### start_task(name: str)

Start tracking a new task. Marks previous task as abandoned if active.

### complete_task(result: str)

Mark current task complete. Calculates duration, saves to state and daily log.

### show_status()

Display current task (if active) and today's completed tasks.

### generate_report()

Print formatted daily report with all tasks, durations, and results.
