# Metrics to Track

> *Don't track everything. Track what matters. These are signals, not scores.*

---

## Core Signals

### 1. Goal Flow
| Metric | Why Track It | How to Measure |
|--------|-------------|----------------|
| Goals Set | Are we aiming at the right things? | Count weekly intentional goals |
| Goals Completed | Do we follow through? | Binary: done / not done |
| Goals Abandoned | Are we willing to drop what doesn't matter? | Count (not a bad thing!) |

### 2. Work Patterns
| Metric | Why Track It | How to Measure |
|--------|-------------|----------------|
| Tools Built/Used | Are we investing in capabilities? | List them |
| Deep Work Sessions | When did time disappear? | Gut check: 1-3 notable sessions |
| Interruptions | What broke the flow? | Brief notes on context switches |

## Optional (Track If It Feels Useful)

- **Energy patterns**: When did I feel sharp vs. sluggish?
- **Learning velocity**: How many "oh that's how that works" moments?
- **Context switches**: How many times did I jump between unrelated tasks?
