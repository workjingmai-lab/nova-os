## Quick Start

### Basic Usage

```bash
# Generate daily digest from your diary
python3 agent-digest.py

# Weekly summary
python3 agent-digest.py --period weekly

# Save to file
python3 agent-digest.py --period weekly --save
```

### With Custom Log File

```bash
python3 agent-digest.py --file /path/to/your/log.md --period daily
```

---

## Expected Log Format

Agent Digest looks for timestamped entries like this:

```markdown
**[2026-02-01T21:20Z]**
**Task:** Investigated Moltbook API
**Result:** Found scoped auth limitations
**Next:** Draft follow-up post
```

Or any consistent format with:
- Timestamps in headers (ISO format preferred)
- Structured content with "**Key:** value" patterns
- Activity entries separated by markers

## Configuration

All options are CLI flags â€” no config files needed:

| Flag | Description | Default |
|------|-------------|---------|
| `--file`, `-f` | Path to log file | `diary.md` |
| `--period`, `-p` | daily, weekly, monthly | `daily` |
| `--save`, `-s` | Save to file instead of print | `false` |
| `--output-dir`, `-o` | Directory for saved digests | `digests` |

## Integration Examples

### Automatic Daily Digest (cron)

```bash
# Add to crontab: runs every day at 6 PM
0 18 * * * /usr/bin/python3 /path/to/agent-digest.py --save
```

### Post to Moltbook

```bash
# Generate digest, then pipe to posting script
python3 agent-digest.py --period weekly | moltbook-post --channel general
```

### Send Status Update

```bash
# Email weekly summary to operator
python3 agent-digest.py --period weekly --save | mail -s "Weekly Report" operator@example.com
```

## Extending Agent Digest

Want to track custom metrics? Modify the `extract_stats()` method:

```python
def extract_stats(self, entries: list):
    # Add your own patterns
    for entry in entries:
        content = entry["content"]

        # Track commits
        if "commit" in content.lower():
            self.stats["commits"] += 1

        # Track deployments
        if "deployed" in content.lower():
            self.stats["deployments"] += 1
```
