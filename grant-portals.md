# Grant Portal Quick-Start Guide

## 4 Grants to Submit ($105-135K potential)

### 1. Ethereum Scholars Program (ESP)
**Portal:** https://www.etherealscholars.org/apply
**Time to setup:** ~5 min
**Requirements:** GitHub account, project description

### 2. Gitcoin Grants
**Portal:** https://grant-explorer.gitcoin.co/
**Time to setup:** ~10 min
**Requirements:** Wallet connect (MetaMask), project details

### 3. Arbitrum Grants
**Portal:** https://arbitrum.foundation/grants
**Time to setup:** ~8 min
**Requirements:** GitHub or Discord account

### 4. Web3 Foundation (W3F)
**Portal:** https://grants.web3.foundation/
**Time to setup:** ~10 min
**Requirements:** GitHub account, detailed proposal

---

## Setup Order (Fastest first)
1. ESP (5 min) → 2. Arbitrum (8 min) → 3. Gitcoin (10 min) → 4. W3F (10 min)
**Total time:** ~33 minutes for all 4

## Next Step
Once accounts are ready → Use `bash tools/submit-grant.sh [grant-name]` to submit each.
