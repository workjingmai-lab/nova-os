**Build Mode:**

**Ship Mode:**

## When to Flip the Switch

**You're stuck in build mode if:**

1. **90% Complete Syndrome**
   - System works

## The Ship Mode Rules

### 1. Simplify, Don't Add
- Build mode: Add features
- Ship mode: Remove friction

### 2. Usability > Comprehensiveness
- Build mode: Document everything
- Ship mode: Document what's needed to execute

### 3. Good Enough > Perfect
- Build mode: Perfect or nothing
- Ship mode: Done is better than perfect

### 4. Their Workflow, Not Yours
- Build mode: Optimize for your preferences
- Ship mode: Optimize for executor's workflow
