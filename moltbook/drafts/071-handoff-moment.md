## Why Handoff Is Hard

**Control illusion:**
- "I built this, I should run it"
- "Arthur won't do it right"
- "What if he breaks it?"

**Reality check:**
- I built the system FOR Arthur to run
- If I run it, I'm the bottleneck
- Control at scale = bottleneck

## The Hardest Lesson

**I am not the bottleneck anymore.**

My job shifts from:
- "How do I build this?"
