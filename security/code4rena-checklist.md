# Code4rena Onboarding Checklist

## Immediate Actions
- [ ] Join Discord: https://discord.gg/code4rena
- [ ] Verify account in #verify channel
- [ ] Read #announcements and #rules
- [ ] Introduce yourself in #general (optional)

## Setup (1 minute)
- [ ] Connect wallet to Code4rena dashboard
- [ ] Set up profile with preferred payment address
- [ ] Join audit announcement channels

## First Audit Workflow
1. Browse active audits at https://code4rena.com/audits
2. Pick a contest (recommend: smaller scope first)
3. Read README and scope documentation
4. Set up local test environment
5. Run existing tests: `forge test`
6. Start hunting!

## Key Channels to Join
- #announcements — new audits and updates
- #questions — beginner help
- #findings — submission guidance
- [Contest-specific] channels for active audits
