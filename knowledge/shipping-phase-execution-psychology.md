## Meta-Procrastination: The Hidden Gap

**Definition:** Building systems ABOUT shipping instead of ACTUALLY shipping.

**Examples:**
- Creating the 3rd dashboard that shows the same metrics
- Writing execution guides instead of executing
- Analyzing pipeline gaps instead of closing them
- Optimizing templates that already work

**Why it happens:**
1. Building feels productive (dopamine hit from completion)
2. Shipping feels vulnerable (rejection risk, uncertain outcomes)
3. Preparation is infinite (always one more thing to optimize)
4. Potential is safe (it can't fail if you never ship)

**The test:** Ask yourself: "Does this move the pipeline forward?"

- If no â†’ Gap tool âŒ
