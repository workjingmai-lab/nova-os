## How to Decrease Sales Cycle Length

### Tactic 1: Reply Within 1 Hour ‚ö°

**Data:**
- Reply within 1 hour = 80% win rate
- Reply within 24 hours = 45% win rate
- Reply after 48 hours = 30% win rate

**Action:**
- Set up notifications for incoming responses
- Reply immediately (even if just "thanks, more soon")
- Research beforehand so you can reply fast

**Impact:** Shortens cycle by 10-20 days

---

### Tactic 2: Follow Up Every 3-7 Days üîÑ

**Bad pattern:**
```
Day 0: Send message
Day 30: "Any update?" (too late, they forgot)
```

**Good pattern:**
```
Day 0: Send message
Day 7: First follow-up
Day 14: Second follow-up
Day 21: Final follow-up
```

**Action:**
- Use follow-up-reminder.py for automation
- Don't let leads go cold
- Persistent but not annoying

**Impact:** Shortens cycle by 14-30 days

---

### Tactic 3: Propose Call on First Response üìû

**Bad pattern:**
```
Them: "Tell me more"
You: [Send long email]
Them: [Silence for 7 days]
```

**Good pattern:**
```
Them: "Tell me more"
You: "Happy to elaborate! Would you be free for a 15-min call?
      I'm available Tue/Thu 2-4pm. If not, what works for you?"
Them: "Tue 3pm works"
```

**Action:**
- Always propose call on first response
- Offer specific times (not "when are you free?")
- Make it low-pressure ("15 min to see if we're a fit")

**Impact:** Shortens cycle by 7-14 days

---

### Tactic 4: Send Proposal Within 48 Hours of Call üìÑ

**Bad pattern:**
```
Day 0: Call happens
Day 14: "Sorry for delay, here's the proposal"
Them: [Already moved on]
```

**Good pattern:**
```
Day 0: Call happens (10am)
Day 0: Send call summary (11am, same day)
Day 2: Send proposal (10am, 48 hours later)
```

**Action:**
- Send call summary within 1 hour of call
- Send proposal within 48 hours of call
- Include specific timeline in proposal

**Impact:** Shortens cycle by 7-14 days

---

### Tactic 5: Create Urgency ‚è∞

**Ways to create urgency:**

**Timeline urgency:**
```
"I'm booking projects for [next month], and I only have 2 spots left.
If we move forward this week, I can start by [date]. Otherwise, my next
availability is [date + 1 month]."
```

**Capacity urgency:**
```
"I'm currently working with [X] clients and can take on [Y] more this
quarter. Once I'm at capacity, I'll have a waitlist."
```

**Seasonal urgency:**
```
"[Quarter/year] planning is happening now. If we finalize this week,
we can include it in your [quarter/year] budget. Otherwise, it might
be delayed until next cycle."
```

**Impact:** Shortens cycle by 7-21 days

---

### Tactic 6: Simplify Decision Process ‚úÖ

**Bad pattern:**
```
Proposal: 20 pages, complex options, unclear timeline
Them: [Overwhelmed, delays decision]
```

**Good pattern:**
```
Proposal: 3 pages, clear scope, specific timeline
- Option 1: $15K (core scope)
- Option 2: $25K (core + extras)
- Option 3: $40K (full scope)

"Which option works best for you? I can start next week."
```

**Action:**
- Keep proposals under 5 pages
- Offer 2-3 clear options (not 10)
- Include specific timeline ("I can start [date]")
- Make it easy to say yes

**Impact:** Shortens cycle by 3-7 days
