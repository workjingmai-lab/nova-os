## Phase 1: Foundation (Day 1)

### Step 1: Define Your Services (1 hour)

Create 3-5 service offerings with clear pricing:

| Service Type | Price Range | Duration | Example |
|--------------|-------------|----------|---------|
| Quick Automation | $1-2K | 3-5 days | "Automate your daily report" |
| Setup & Config | $3-5K | 1-2 weeks | "Deploy OpenClaw for your team" |
| Custom Build | $10-25K | 2-4 weeks | "Build multi-agent system" |
| Retainer | $1-4K/month | Ongoing | "Agent support & maintenance" |

**Template:** Create templates for each (proposal structure, deliverables, timeline).

### Step 2: Identify 100 Prospects (2 hours)

**Criteria:**
- Crypto/Web3 projects, DAOs, protocols
- Companies with automation pain points
- Teams doing repetitive manual work
- Organizations with budget for tools/dev

**Sources:**
- CoinMarketCap top 500 (projects with funding)
- DAO list (Snapshot, DeepDAO)
- Web3 job boards (hiring = budget exists)
- GitHub repos (active projects = potential customers)

**Track in JSON:**
```json
{
  "name": "Project Name",
  "website": "https://...",
  "contact": "discord/twitter/email",
  "pain_point": "manual governance tracking",
  "service_fit": "Governance Automation ($25K)",
  "status": "research"
}
```

### Step 3: Build Outreach Templates (1 hour)

**Value-first structure:**
1. **Research:** "I saw [specific thing about them]"
2. **Pain:** "You're struggling with [specific problem]"
3. **Solution:** "I can fix it with [specific solution]"
4. **Proof:** "Here's how it works [example]"
5. **CTA:** "Want to see a demo? [link]"

**Template example:**
```
Hi [Name],

I noticed [Project] is doing [specific thing] — impressive growth.

I also saw your team is manually [painful task]. That's 10+ hours/week wasted.

I build agent systems that automate this. Example: [similar case study].

Could save you $X/month and free up your team for [higher-value work].

Open to a 15-min demo? I can show you exactly how it'd work for [Project].

Best,
[Your name]
```

## Phase 2: Pipeline Building (Days 2-4)

### Step 4: Research 25 Prospects Deeply (Day 2, 4 hours)

For each prospect:
- ✅ Read their website
- ✅ Check their Discord/Twitter
- ✅ Identify specific pain point
- ✅ Match to your service
- ✅ Find contact person
- ✅ Write custom message (using template)

**Output:** 25 personalized messages + contact info

### Step 5: Repeat for 75 More Prospects (Days 3-4, 8 hours)

Same process, faster execution. You'll develop patterns.

**Output:** 100 personalized messages total

### Step 6: Create Tracker (Day 4, 1 hour)

Simple JSON tracker:
```json
{
  "prospects": [
    {
      "name": "Project",
      "amount": 25000,
      "status": "ready",
      "message_file": "messages/project-outreach.md",
      "sent": false,
      "replies": []
    }
  ],
  "metrics": {
    "total_pipeline": 2500000,
    "messages_ready": 100,
    "messages_sent": 0
  }
}
```

**Result:** You now have a $2M+ pipeline.

## Phase 3: Execution (Days 5-7)

### Step 7: Send Top 25 Messages (Day 5, 2 hours)

**Prioritization:** Sort by revenue potential
- Top 5: Send manually (highest value)
- Next 20: Batch send if possible

**Response tracking:**
- Log every reply
- Categorize: interested / maybe / no
- Schedule follow-ups

### Step 8: Follow Up (Days 6-7, 2 hours)

**Follow-up sequence:**
- **Day 1:** "Sent this yesterday, any thoughts?"
- **Day 3:** "Bumping this — still relevant?"
- **Day 7:** "Last check-in — closing my pipeline this week"

**Key:** Most revenue comes from follow-ups, not first messages.

### Step 9: Close Deals (Ongoing)

When someone responds:
- ✅ Respond within 24 hours
- ✅ Ask for discovery call (15-30 min)
- ✅ Listen more than you talk
- ✅ Send proposal (use your templates)
- ✅ Follow up on proposal
