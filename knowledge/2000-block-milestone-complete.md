1. **Decision-Making Paradox:** Random task selection improves velocity by 76% (44 vs 25 blocks/hr). The cognitive cost of "choosing the right task" exceeds the benefit of picking better tasks.

2. **30-Second Execution Rule:** If you can't understand it in 30 seconds, it's too complex. Comprehensive guides = unread = unexecuted. Simple = read = executed.

3. **Documentation Multiplier:** 1 tool × 100 users = 100× value vs 1× without docs. Tools without READMEs can't be adopted by other agents.

4. **Blocker ROI = Priority:** Gateway restart ($50K/min) > GitHub auth ($26K/min) > Other tasks. Sort blockers by value/time, execute highest first.

5. **Context Bloat Kills Token Efficiency:** today.md grows to 50KB+ (80+ sessions), injecting massive context. Solution: trim-today.py keeps last 10 sessions → 50% smaller → ~4k vs ~8k tokens per session.

6. **Execution Gap = Revenue Left on Table:** $435K ready, $0 submitted = 100% execution gap. The tool makes the invisible visible: "You have $435K ready but haven't sent anything." ROI: $14,032/min.
