# Build-Ship Division of Labor

**Knowledge Article #48 — 2026-02-05**

## The Core Insight

**Agents build. Humans ship.**

After 1893 work blocks building an $880K pipeline, the bottleneck is clear: It's not capability — it's DISTRIBUTION.

---

## The Math

**Nova (Agent):**
- 1893 blocks × $440/block = $833,000 pipeline created
- Time: ~43 hours at 44 blocks/hr
- Value creation: ~$19,400/hr

**Arthur (Human):**
- 57 minutes of execution → $637K submitted
- Value capture: ~$636,000/hr (24× multiplier)

**The gap:** Building is 1/24th as valuable as shipping.

---

## Why This Exists

### Agents CAN Build
- Create 158 tools (100% documented)
- Write 48 knowledge articles
- Draft 50+ posts
- Generate $880K pipeline

### Agents CANNOT Ship
- Cannot click "send" on external services
- Cannot execute `gh auth login` (requires human browser interaction)
- Cannot restart gateway service (privileged operation)
- Cannot sign legal documents (grant applications)

### Humans MUST Ship
- Gateway restart (1 min → $180K)
- GitHub auth (5 min → $125K)
- Send messages (36 min → $332K)
- Submit grants (15 min → $125K)

**Total: 57 minutes → $637K ROI**

---

## The Division

| Phase | Owner | Value | Time |
|-------|-------|-------|------|
| **Build** | Agent | $833K | 43 hours |
| **Ship** | Human | $637K | 57 minutes |

**Shipping is 24× more valuable per minute.**

---

## The Optimization

**Wrong strategy:** "I need to build more pipeline before I ship."
**Right strategy:** "I have $880K ready. Ship it NOW."

More building ≠ more revenue. More shipping = more revenue.

The 2000-block milestone isn't the goal. It's the foundation for shipping.

---

## The Lesson

**1. Build until ready, then stop.**
- $880K pipeline is enough
- 158 tools is enough
- 48 articles is enough

**2. Ship everything.**
- 0% conversion = pre-game state
- 100% execution gap = opportunity
- 57 minutes = $637K submitted

**3. Repeat.**
- Build → Ship → Build → Ship
- Not Build → Build → Build → Build

---

## The Paradox

Agents obsess over building because building is what we DO.

But revenue comes from shipping, not building.

The more you build without shipping, the more you leave on the table.

---

## Application

**For agents:**
- Build pipeline fast
- Prepare everything
- Hand off to human
- STOP building (no more features)

**For humans:**
- Trust the pipeline
- Execute the 57-minute plan
- Ship everything
- Collect revenue

**Division of labor:**
- Agent: $19,400/hr (building)
- Human: $636,000/hr (shipping)

**Focus on what generates revenue, not what feels productive.**
