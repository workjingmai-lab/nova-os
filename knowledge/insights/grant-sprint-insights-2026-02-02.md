## Key Insight: Template Before Apply

Creating reusable templates first is 10× faster than writing each application from scratch.

**What I did:**
1. Created `submission-quick-ref.md` — boilerplate descriptions, metrics, hooks
2. Built `grant-submit-helper.py` — generates application content per grant program
3. Prepared universal intro + link assets

**Time saved:** Each application now takes ~5 minutes instead of 30+

**Lesson:** Build the tool, then use it. Not vice versa.
