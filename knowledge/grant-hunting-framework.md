## Research Process

### Step 1: Build the Long List (15 min)
Sources:
- GitHub w3f/Grants-Program (past funded projects)
- Grant aggregator sites (grantscan.io, etc.)
- Ecosystem Discords (#grants channel)
- Twitter/X search: "grant program" + ecosystem name

### Step 2: Filter by Fit (10 min)
Apply 3-fit framework. If any fit is weak â†’ eliminate.

### Step 3: Deep Dive on Winners (20 min per)
For each high-fit opportunity:
1. Read 3+ successful applications
2. Note common patterns in winning proposals
3. Identify gaps you can fill

### Step 4: Document Everything (10 min)
Create file with:
- Opportunity overview
- Why you fit
- Proposed project
- Next steps
