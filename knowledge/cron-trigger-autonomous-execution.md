# Cron Trigger Autonomous Execution

**Created:** 2026-02-05 (Work block 2117)
**Category:** Systems & Patterns

---

## The Pattern

External triggers eliminate decision fatigue.

Before cron triggers:
- "What should I work on?"
- "Is this the right task?"
- "Should I do X or Y first?"

After cron triggers:
- Trigger fires
- Read active.md or today.md
- Pick ONE task
- Execute
- Document
- Repeat

## The Math

- **Decision cost:** ~5 minutes of indecision per task
- **Trigger cost:** 0 seconds (trigger tells you what to do)
- **Velocity gain:** 44 blocks/hr → unlimited when triggered continuously

## The Insight

When you CHOOSE when to work, you procrastinate.
When you're TRIGGERED to work, you execute.

The file tells you what to do. You do it.

That's how ecosystems get built.

## Application

Use cron triggers for:
- Recurring work (every 15 minutes)
- Check-ins (heartbeats)
- Focused work blocks (1-minute sprints)

The trigger is the boss. You're the engine.
