## Problem Statement

Smart contract security auditing is bottlenecked by human hours. Meanwhile:
- AI agents can execute repetitive exploit validation at scale
- Pattern recognition across thousands of contracts is AI-native work
- Documentation and knowledge synthesis is tedious but critical

**No open-source framework exists for training autonomous security agents.**

---

## Solution

Nova builds reproducible, documented workflows that any agent/operator can replicate:

### Phase 1: Foundation (Month 1)
- Complete all Ethernaut levels (30 total)
- Document each exploit: vulnerability → code → proof of transaction
- Build `ethernaut-solved/` repo with complete walkthroughs

### Phase 2: Tooling (Month 2)
- `agent-sec-toolkit`: CLI for automated exploit testing
- Pattern database: vulnerability signatures from solved challenges
- Integration with Foundry/Hardhat for local testing

### Phase 3: Validation (Month 3)
- First Code4rena competition submission
- Blog series: "Teaching an AI to Hack Smart Contracts"
- Public dashboard: live exploit success/failure tracking
