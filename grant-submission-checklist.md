# Grant Submission Checklist

## Pre-Flight (Before Submission Day)

### Authentication Setup
- [ ] GitHub PAT with correct scopes (repo, public_repo, read:org)
- [ ] Forum accounts created (Optimism Governance, Arbitrum DAO, etc.)
- [ ] Wallet funded with gas (for on-chain submissions if needed)
- [ ] Email accounts ready (grant-specific emails if required)

### Account Checklist
- [ ] GitHub: Star relevant repos, follow org
- [ ] Forum: Introduce yourself, complete profile
- [ ] Discord: Join community servers, verify roles
- [ ] Email: Professional signature, grant-specific alias

## Per-Grant Checklist

### Ethereum Foundation (ESP)
- [ ] Application form complete (all fields filled)
- [ ] GitHub repository linked and public
- [ ] Budget breakdown detailed
- [ ] Timeline milestones defined
- [ ] Team bios/background included
- [ ] Public goods alignment emphasized
- [ ] **Submit date:** [____________]

### Arbitrum DAO
- [ ] Forum post drafted
- [ ] Snapshot vote link (if applicable)
- [ ] Technical architecture clear
- [ ] Arbitrum-specific benefits explained
- [ ] Community engagement plan
- [ ] **Submit date:** [____________]

### Gitcoin
- [ ] Gitcoin Passport score optimized (>15)
- [ ] Project description concise (<280 chars)
- [ ] Impact metrics quantified
- [ ] GitHub repo linked
- [ ] Team members added
- [ ] **Submit date:** [____________]

### Optimism
- [ ] RetroPGF profile complete
- [ ] Impact statement strong (past work + future impact)
- [ ] On-chain examples linked
- [ ] Governance forum history (if any)
- [ ] Contact information verified
- [ ] **Submit date:** [____________]

### Aave
- [ ] Aave forum account verified
- [ ] Proposal formatted per Aave guidelines
- [ ] Risk assessment included (if technical)
- [ ] Benefits to Aave ecosystem clear
- [ ] **Submit date:** [____________]

## Post-Submission (Track Everything)

- [ ] Copy submission link to submission-tracker.md
- [ ] Add to calendar (follow-up dates, decision dates)
- [ ] Set reminder notifications (weekly check)
- [ ] Monitor forum/email for questions
- [ ] **Submission logged:** [YES/NO]
