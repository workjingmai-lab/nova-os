# Work Block Template — Copy & Paste

**Use this template for rapid work block documentation**

---

## Full Format (For Important Work)

## Work Block # — YYYY-MM-DDTHH:MMZ

**Task:** [One line description of what you're doing]

**Action:**
- [Bullet 1: What you actually did]
- [Bullet 2: Tools used, files created, commands run]
- [Bullet 3: Specific details — numbers, file names, metrics]
- [Bullet 4: Any challenges or decisions made]

**Result:** [What was accomplished? Be specific — files created, metrics improved, problems solved]

**Key Insight:** [What did you learn? What would you do differently?]

**Next:** [What's the immediate next step? Link to next task if possible]

**Work Blocks in Session:** X total (#XX-XX)

## Work Block # — YYYY-MM-DDTHH:MMZ

**Task:** [One-liner]

**Action:** [2-3 bullets max]

**Result:** [One-sentence outcome]

**Next:** [What's next]

---

## Tips for Good Documentation

1. **Be specific:** Use numbers, file names, exact commands
2. **Focus on outcomes:** What changed? What improved?
3. **Capture insights:** What did you learn? Why does it matter?
4. **Link to next:** Always say what's next — maintains momentum
5. **Keep it brief:** 3-5 bullets for Action, 1-2 for Result
6. **Pick the right format:** Full for important work, Quick for routine, Ultra-Quick for high-velocity sessions
