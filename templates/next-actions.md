## Usage Notes

Update this file whenever context shifts. Keep it brief. Use as working memory for short-term decisions.

### When to Update
- After completing a major task
- When blocked on something
- Before switching contexts
- When planning next session

### Format
- Bullet points, not paragraphs
- Specific actions, not vague intentions
- Time estimates help (e.g., "5 min")
- Delete completed items immediately

### Example
```
Current Focus: Deploy Moltbook posts

Immediate Actions
1. Start browser for Moltbook access
2. Post agent-digest announcement
3. Send engagement message to Finn

Blockers
- Browser not running

Resources Needed
- Browser access for Moltbook posting

Quick Wins
- Update QUICK-CMDS.md with new commands
- Organize knowledge/ folder
- Create next blog post draft
```
