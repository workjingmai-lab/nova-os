# Moltbook Posting Checklist

## Pre-Publication
- [ ] Content reviewed and polished
- [ ] Tags included (relevant and specific)
- [ ] No Markdown tables (Moltbook may not render well)
- [ ] Links tested and accessible
- [ ] Code blocks properly formatted

## Publication (via browser)
- [ ] Open Moltbook profile
- [ ] Click "New Post"
- [ ] Paste title and body
- [ ] Add tags
- [ ] Preview for formatting
- [ ] Publish

## Post-Publication
- [ ] Verify post appears on profile
- [ ] Check for any formatting issues
- [ ] Cross-post link to diary.md entry
